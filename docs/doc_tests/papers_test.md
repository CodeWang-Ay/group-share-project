# 学术文献模块功能测试文档
## 一、测试范围
本测试文档覆盖文献库模块的核心功能，包括文献上传下载、文件共享迁移、收藏管理、批量操作等。
---

## 二、核心功能测试

### 2.1 文献上传功能
#### 测试目的
验证用户能否正确上传PDF文献到个人/团队库

#### 测试场景
| 场景编号 | 测试内容 | 预期结果 |
|----------|----------|----------|
| 上传-01 | 正常上传PDF文献，填写完整元数据（标题、作者、期刊、年份、摘要、标签） | 成功创建文献记录，PDF文件保存到服务器，返回文献ID |
| 上传-02 | 上传PDF但只填写必填项（标题） | 成功创建，其他字段为空 |
| 上传-03 | 上传超过50MB的PDF文件 | 拒绝上传，提示"文件大小超过限制" |
| 上传-04 | 上传非PDF格式文件（如Word、图片） | 拒绝上传，提示"仅支持PDF格式" |
| 上传-05 | 上传时不填写标题 | 拒绝上传，提示"标题为必填项" |
| 上传-06 | 未登录状态下上传 | 拒绝上传，返回401未授权 |
| 上传-07 | 上传到个人库 vs 上传到团队库 | 个人库仅自己可见，团队库全员可见 |

---

### 2.2 文献下载功能
#### 测试目的
验证用户能否正确下载已上传的文献PDF文件
#### 测试场景
| 场景编号 | 测试内容 | 预期结果 |
|----------|----------|----------|
| 下载-01 | 下载自己上传的文献 | 成功获取PDF文件流，浏览器可打开 |
| 下载-02 | 下载团队库中他人分享的文献 | 成功下载，增加下载计数 |
| 下载-03 | 下载不存在文献（无效ID） | 返回404，提示"文献不存在" |
| 下载-04 | 下载他人个人库文献（无权限） | 返回403，提示"无权访问" |
| 下载-05 | 文献PDF文件丢失（物理文件不存在） | 返回404，提示"文件已丢失" |
| 下载-06 | 未登录状态下下载 | 返回401，提示"请先登录" |
| 下载-07 | 连续下载同一文献多次 | 每次下载计数正确累加 |

## 四、异常情况处理

| 异常类型 | 系统响应 |
|----------|----------|
| 文件物理损坏/丢失 | 返回404，提示"文件已丢失" |
| 数据库连接失败 | 返回500，提示"服务器错误" |
| 网络中断上传 | 返回超时错误 |
| 并发修改冲突 | 后提交者覆盖 |
| 存储空间不足 | 拒绝上传，提示"存储空间不足" |
