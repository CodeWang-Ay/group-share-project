# CLAUDE.md

此文件为 Claude Code (claude.ai/code) 在此代码库中工作时提供指导。

## 项目概述

这是一个**研究生组会文件共享系统** - 一个基于 Web 的平台，用于管理研究组会、文件共享和学术协作。该系统专为研究实验室设计，用于协调组会、共享研究材料和跟踪学生进度。

## 技术栈

- **前端**: 纯 HTML5 配合嵌入式 JavaScript 和 Tailwind CSS (通过 CDN)
- **样式**: Tailwind CSS 配合自定义配色方案和工具类
- **图标**: Font Awesome 4.7.0
- **图片**: Picsum Photos API 占位图片
- **架构**: 具有客户端交互性的静态 Web 应用程序

## 核心功能

### 主仪表板 (index.html)
- 导师/教授角色界面
- 即将到来的组会管理
- 最近提交材料跟踪
- 文献库管理
- 学生进度监控
- 存储空间统计
- 团队成员管理

### 认证页面
- **login.html**: 多字段登录（用户名/学号/邮箱）配验证码
- **register.html**: 基于角色的注册（老师/学生）配验证

### UI/UX 模式
- 使用 Tailwind CSS 网格系统的响应式设计
- 自定义配色方案（主色: #2563eb, 次色: #10b981）
- 交互式侧边栏导航
- 基于卡片的内容组织布局
- 实时反馈的表单验证
- 模态框提醒和通知

## 开发工作流

### 运行应用程序
这是一个静态 Web 应用程序 - 只需在 Web 浏览器中打开任何 HTML 文件：
```bash
# 打开主仪表板
open index.html

# 或登录页面
open login.html

# 或注册页面
open register.html
```

### 本地开发
```bash
# 本地提供应用服务（需要 Python 3）
python -m http.server 8000
# 然后访问 http://localhost:8000

# 或使用 Node.js serve（如果可用）
npx serve .
# 然后访问 http://localhost:3000
```

### 无需构建过程
- **无需编译**: 纯 HTML/CSS/JavaScript
- **无依赖项**: 所有外部资源通过 CDN 加载
- **无需打包**: 每个 HTML 文件都是自包含的

## 代码架构

### 组件结构
- **模块化 CSS**: 配合自定义组件的 Tailwind 工具类
- **组件类**: 可重用模式如 `.sidebar-icon`、`.file-card`、`.meeting-card`
- **JavaScript 模块**: 嵌入式脚本中的事件处理和表单验证

### 数据流
- **仅客户端**: 目前无后端集成
- **表单处理**: JavaScript 验证和提交阻止
- **状态管理**: 简单的 DOM 操作和类切换

### 样式架构
- **Tailwind 基础**: 响应式工具和布局系统
- **自定义主题**: 扩展的调色板和组件样式
- **响应式设计**: 移动优先的方法和断点
